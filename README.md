# Booth Blackjack (Koyeb)

## Local run
1) Set DATABASE_URL env var (Postgres)
2) npm install
3) npm start

Static files are in /public.
Put your assets into /public/assets:
- bg.png, chip.png, bet.mp3, throw.mp3, flip.mp3, win.mp3, lose.mp3, shuffle.mp3

Put your card images into /public/cards:
- back.png
- 0.png ... 7.png  (or whatever ranks you use)
